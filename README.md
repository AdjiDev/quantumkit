**Setup guide**
- requires python 3.11 or later

**installation**
- clone this repo first
  ```sh
  git clone https://github.com/AdjiDev/quantumkit
  ```
- install python module
```sh
cd quantum kit
pip install module.txt
```
- run
```
python adji.py
```
